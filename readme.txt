WPC Order Notes help you manage all order notes more easily. You can see all notes in one place and quickly view notes of an order in the popup.

== Description ==

**WPC Order Notes for WooCommerce** is an easy tool for store owners to manage all order notes. It offers a quick preview of added notes in each order from the popup. It lists all the notes with paging and provides the search function for finding the needed detail quickly.

**WPC Order Notes for WooCommerce** makes life easier for store managers by reducing the time needed for organizing notes, checking orders to find the latest activity, or navigating back and forth to check the progress during the package delivery.

= Features =

- List all notes with paging
- Search for matching notes in the Notes section
- Preview the latest notes from the Orders section
- Quick view all notes of an order from the popup
- Add/ edit/ delete a note on the quick view popup

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Order Notes", Install & Activate it
4. Now you can open the WooCommerce/Notes page to see all notes
